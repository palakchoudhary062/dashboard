## Requirements

- Node.js version >= 18.0.0
- npm version >= 9.0.0
- nodemon version >= 3.0.0

## Getting Started

To start the application, follow these steps:

1. Open one terminal.
2. npm install
3. npm run server
4. Open another terminal.
5. npm run client
6. Visit http://localhost:3000/ in your browser.
